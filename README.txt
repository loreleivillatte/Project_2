Ceci est un exercice qui a pour objectif de mettre en pratique les notions que j'ai acquises dans les deux premières parties de ce cours sur Git et Github.

Pour ce faire je vais créer un repository Git simple contenant :

- Un fichier nommé "README.txt" qui présente le projet ;
- 2 autres fichiers de mon choix ;
- Au moins 4 commits distincts montrant l’ajout puis la modification de certains fichiers.
